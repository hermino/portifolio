# Hermino Freitas - Frontend Developer Portfolio

Portfolio website built with **Angular 21** showcasing my experience, projects, and skills as a Frontend Developer.

## 🚀 Technologies

- **Angular 21.1** - Modern frontend framework
- **TypeScript** - Type-safe development
- **CSS3** - Custom styling with design tokens
- **JetBrains Mono** - Monospace typography
- **Responsive Design** - Mobile-first approach
- **Dark/Light Theme** - User preference support

## 🎨 Features

- ✅ Fully responsive design
- ✅ Dark/Light theme toggle
- ✅ SEO optimized with meta tags and structured data
- ✅ Accessible (WCAG compliant)
- ✅ Fast performance
- ✅ Component-based architecture

## 🛠️ Development

### Prerequisites

- Node.js (v22+ recommended)
- npm 10.9+
- Angular CLI 21.1+

### Installation

```bash
# Clone the repository
git clone https://github.com/hermino/hermino.github.io.git
cd hermino.github.io

# Install dependencies
npm install
```

### Running locally

```bash
# Start development server
npm start

# Visit http://localhost:4200
```

### Build

```bash
# Production build
npm run build

# Build for GitHub Pages
npm run build:gh-pages
```
